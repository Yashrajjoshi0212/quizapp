Quiz Learning App 🎯

A simple and interactive Quiz Learning App built using HTML, CSS, and JavaScript. This project helps users test their knowledge through multiple-choice questions while providing instant feedback and score tracking.

📌 Features

🧠 Multiple-choice quiz questions

⚡ Instant answer feedback

📊 Score tracking

🔁 Next question navigation

📱 Responsive design

🎨 Clean and simple UI

🛠️ Built With

HTML5 – Structure of the application

CSS3 – Styling and responsive design
